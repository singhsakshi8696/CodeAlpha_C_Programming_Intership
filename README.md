## 🚀CodeApha_C_Internship

This repository houses the projects I built during my **C Programming Internship at CodeAlpha**. The internship focused on strengthening foundational mastering procedural programming concepts, and understanding memory and file management in C.

### 🛠️ Technical Skills Acquired & Demonstrated:

* **Data Structures:** Implementation of 2D Arrays and Custom Structures (`struct`).
* **Data Persistence:** File handling (`fopen`, `fwrite`, `fread`) to read and write permanent data to local storage.
